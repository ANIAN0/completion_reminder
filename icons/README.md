# 图标说明

本目录应包含Chrome插件图标文件。

## 所需图标尺寸：
- 16x16: `icon16.png`
- 32x32: `icon32.png`
- 48x48: `icon48.png`
- 128x128: `icon128.png`

## 图标设计建议：
- 简约风格，与shadcn/ui保持一致
- 建议使用对话气泡或AI相关的图标
- 颜色建议：
  - 主色：#0f172a（深蓝色）
  - 背景：白色或透明
  - 边框：#e2e8f0（浅灰色）

## 临时方案：
如果您暂时无法创建图标文件，可以从以下资源获取：
1. [IconFinder](https://www.iconfinder.com/)
2. [Flaticon](https://www.flaticon.com/)
3. [Heroicons](https://heroicons.com/)

请确保使用的图标有商业使用许可。
